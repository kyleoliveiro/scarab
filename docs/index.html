    [private] __SCARAB

    $__SCARAB: (
  'CONSTANTS' : (),
  'INVENTORY' : (),
  'OPTIONS'   : ()
);
    View source

    Description

    Global variable where all user configuration is stored.

    Never modify this variable or its children directly. Instead, use:

    • set-option() / option() to manage options
    • define() / const() to manage constants
    • set() / get() to manage the inventory

    Type

    Map

    Map structure

    Map keyNameMap keyDescriptionMap keyTypeMap keyValue
    OPTIONS

    Contains Scarab options

    		Map—none
    CONSTANTS

    Contains constant values

    		Map—none
    INVENTORY

    Contains the stylesheet inventory

    		Map—none

    • [function] set-option

    Description

    The global stylesheet inventory.

    Inventory values can be configured using set(). They can be retrieved using get(), or any of the inventory helper functions like font-size(), fz(), etc.

    During initialization, the $__SCARAB_INVENTORY variable is merged into $__SCARAB. Subsequently, its value is set to null.

    Description

    Options for the Scarab framework.

    Scarab options can be configured using set-option(). Refer to the table below for a description of each option.

    During initialization, the $__SCARAB_OPTIONS variable is merged into $__SCARAB. Subsequently, its value is set to null.

    Type
